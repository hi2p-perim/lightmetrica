Detailed Description

template<typename T>
struct TMat4< T >

4x4 matrix. Generic column major 4x4 matrix. A matrix v00 v01 v02 v03 v10 v11 v12 v13 v20 v21 v22 v23 v30 v31 v32 v33 is stored sequentially as v00, v10, ..., v33.

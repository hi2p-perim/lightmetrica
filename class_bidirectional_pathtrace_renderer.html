Detailed Description

Veach's bidirectional path trace renderer. An implementation of bidirectional path tracing (BPT) according to Veach's paper. Reference: E. Veach and L. Guibas, Bidirectional estimators for light transport, Procs. of the Fifth Eurographics Workshop on Rendering, pp.147-162, 1994.

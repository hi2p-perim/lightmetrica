Detailed Description

Path tracing with direct light sampling. Implements path tracing with direct light sampling (a.k.a. next event estimation). In this implementation, E{D,S}D+L paths are sampled by direct light sampling and E{D,S}*S+L paths are sampled by BSDF sampling.
